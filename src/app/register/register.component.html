How to play

En esta parte se describen problemas y dilemas enfrentados durante la creación de este proyecto.

Gran parte del esfuerzo en este proyecto se enfocó en crear workarounds alrededor de Angular y sus restricciones de uso de JS. Sin embargo, las restricciones resultaron ser mayores en algunos casos y se tuvo que utilizar Angular de todos modos.

Se intentó generar una base para un juego, pero está lejos de estar completo. No existe sistema de palabras y el sistema de dibujo es notoriamente deficiente: entrega cada vez todo el dibujo en vez de solo los cambios, esto para evitar utilizar la base de datos como sistema de información para el juego y no eliminar partes del dibujo para otros jugadores.

Sin embargo, se logro solucionar decenas de problemas e incluso aprender sobre todas las tecnologías utilizadas en el proyecto. Incluso Angular.
